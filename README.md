### Clone project

```sh
git clone git@github.com:tlos3r/cocktail-info.git
```

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
